Guia de compilacion:
VR: Para hacer el build de VR asegurarse que el plugin openXR este habilitado y que android manifest en la seccion de Player este desactivado
Android: Para hacer build tiene q desactivar el plugin openXR y habilitar el android manifest

scenes:
Vr: inicio + Level1
Android: New Scene

El proyecto no compila bien si se ejecuta con el play de unity por un fallo en el que por alguna razon se destruye pero al hacer el build si se aplican sus respectivas camaras para cada uno.

El proyecto en unity parece nuevo pero eso es debido a que nosotros usamos Unity Control Version ya que nos fue mas comodo y estaba mas integrado con el motor
